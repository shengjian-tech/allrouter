AllRouter.AI — это единый шлюз больших языковых моделей, созданный для удобного, эффективного и экономичного доступа к AI-моделям. Это руководство описывает ключевые функции и сценарии использования с точки зрения обычного пользователя.

# Обзор платформы

AllRouter.AI объединяет более 50 популярных моделей, включая OpenAI, Claude и Llama. Через единый API вы получаете доступ к полному набору AI-возможностей без переключения между разными платформами.

Ключевые преимущества:

Бесшовная замена: 100% совместимость с OpenAI SDK, достаточно изменить `base_url`.

Умная маршрутизация: автоматический выбор более дешёвой или более быстрой модели по заданным правилам.

Высокая доступность: автоматическое переключение при сбоях для непрерывной работы.

Наблюдаемость в реальном времени: в консоли доступны подробные метрики задержки, расхода токенов и стоимости.

# Быстрый старт

## Регистрация и вход

### Регистрация

Откройте https://allrouter.shengjian.net/, нажмите **Вход** -> **Регистрация**, введите имя пользователя и пароль во всплывающем окне и нажмите **Регистрация**.

![](/docs_images/img_2.png)

![](/docs_images/img_3.png)

### Вход

Перейдите на https://allrouter.shengjian.net/, нажмите **Вход**, введите имя пользователя и пароль, затем нажмите **Продолжить**, чтобы войти в AllRouter.AI.

![](/docs_images/img_4.png)

## Песочница

Описание: тестируйте взаимодействие с AI-моделями прямо в веб-интерфейсе без написания кода.

1. Шаг 1: в левом меню нажмите **Песочница**.

2. Шаг 2: в панели **Настройки модели** выберите модель в списке **Модель** (например, `gpt4o`).

3. Шаг 3: (опционально) настройте ползунок **Temperature** для управления случайностью ответа.

4. Шаг 4: введите запрос в поле ввода внизу правой области чата.

5. Шаг 5: нажмите кнопку **Отправить** справа от поля ввода.

Предусловие: баланс аккаунта должен быть больше 0.

Ожидаемый результат: AI отвечает сразу, а внизу отображаются детали расхода.

![](/docs_images/img_5.png)

## Консоль

### Панель мониторинга

После входа в консоль сначала открывается дашборд. Он наглядно показывает состояние аккаунта:

Текущий баланс: просмотр оставшихся средств в реальном времени.

Статистика использования: количество запросов, списанный баланс и расход токенов.

Потребление ресурсов: общий расход средств и токенов.

Метрики: средние TPM и RPM.

Аналитика по моделям: графики распределения расходов и трендов вызовов.

![](/docs_images/img_6.png)

### Управление токенами

Описание: управление API-ключами (токенами), лимитами и сроком действия.

Чтобы начать API-вызовы, создайте токен:

1. Шаг 1: нажмите **Управление токенами** в левом меню.

2. Шаг 2: нажмите **Добавить токен**.

3. Шаг 3: задайте имя токена, лимит, срок действия и ограничения доступа.

4. Шаг 4: после создания скопируйте сгенерированный ключ и используйте его в приложении.

![](/docs_images/img_7.png)

### Журналы использования

Описание: фиксирует все API-вызовы через шлюз AllRouter.AI для аудита и анализа затрат.

1. Шаг 1: нажмите **Журналы использования** в левом меню, чтобы открыть список вызовов.

2. Шаг 2: задайте диапазон времени через выбор дат (**начало** и **конец**).

3. Шаг 3: введите фильтры в поля **Имя токена**, **Имя модели**, **Группа** или **Request ID**.

4. Шаг 4: нажмите **Поиск**, чтобы обновить список.

5. Шаг 5: просматривайте **Время**, **Токен**, **Модель**, **Стоимость** и открывайте **Детали** для полного запроса/ответа.

![](/docs_images/img_8.png)

### Журналы генерации изображений

Описание: отслеживание задач генерации изображений (например, Midjourney): статус, прогресс и результат.

1. Шаг 1: в левом меню нажмите **Журналы изображений**.

2. Шаг 2: найдите задачу по **Task ID**, выберите период и нажмите **Поиск**.

3. Шаг 3: откройте результат по миниатюре в колонке **Изображение результата**.

4. Шаг 4: при ошибке смотрите причину в колонке **Причина ошибки**.

![](/docs_images/img_9.png)

### Журналы задач

Описание: история жизненного цикла всех асинхронных задач (пакетная обработка, долгие задачи и т.д.).

1. Шаг 1: нажмите **Журналы задач** в левом меню.

2. Шаг 2: отфильтруйте по **Task ID** или по времени, затем нажмите **Поиск**.

3. Шаг 3: сравните **Время отправки** и **Время завершения** для анализа длительности.

4. Шаг 4: настройте видимые столбцы через **Настройка столбцов** справа вверху.

![](/docs_images/img_10.png)

## Личный кабинет

### Управление кошельком

Описание: управление балансом, поддержка нескольких способов оплаты и реферальных бонусов.

1. Шаг 1: в левом меню нажмите **Управление кошельком**.

2. Шаг 2: введите сумму пополнения (USD) в поле **Сумма пополнения**.

3. Шаг 3: выберите способ оплаты: **WeChat** или **Stripe**.

Предусловие: доступен действующий способ оплаты.

Ожидаемый результат: после оплаты поле **Текущий баланс** обновляется в реальном времени.

# Витрина моделей

Описание: **Витрина моделей** — центральный каталог AllRouter.AI. Здесь показаны все поддерживаемые модели и доступен прозрачный просмотр тарифов в реальном времени. По официальным коэффициентам или фактической стоимости после пополнения можно точно оценить стоимость каждого вызова.

1. Шаг 1: в верхней навигации нажмите **Витрина моделей**. По умолчанию откроется карточный вид с ID модели и ценой за 1M токенов.

2. Шаг 2: разберитесь в двух режимах тарификации.

* Тарификация по токенам: для текстовых/чат-моделей, оплата за входные, выходные и кэш-читанные токены.
* Тарификация по запросам: для некоторых моделей (рисование, задачи) фиксированная цена за запрос.

3. Шаг 3: переключайте режим отображения цены.

* Цена после пополнения: показывает фактическое списание с учётом вашего коэффициента пополнения.
* Отображение коэффициента: показывает тарифный коэффициент относительно официальной цены.

4. Шаг 4: точно находите модели через боковую панель: поставщик и тип тарификации.

5. Шаг 5: используйте поиск и копирование ID модели через иконку **Копировать**.

6. Шаг 6: пакетные операции и смена вида.

* Табличный вид: нажмите **Табличный вид** справа вверху для компактного сравнения цен.
* Ожидаемый результат: значительно повышается эффективность настройки в мульти-модельных сценариях.

# Пример использования

Пример: использование AllRouter в Claude Code.

## Шаг 1: установка Claude Code

Предусловия:

* Установите [Node.js 18 или новее](https://nodejs.org/en/download/).
* Для macOS рекомендуется установка через [nvm](https://github.com/nvm-sh/nvm) или [Homebrew](https://formulae.brew.sh/formula/node).
* Для Windows также нужен [Git for Windows](https://git-scm.com/download/win).

В терминале выполните:

npm install -g @anthropic-ai/claude-code

Проверьте установку:

claude --version

## Шаг 2: настройка AllRouter

1. Шаг 1: зарегистрируйте аккаунт

Откройте платформу AllRouter, нажмите **Регистрация/Вход** в правом верхнем углу и завершите регистрацию.

2. Шаг 2: получите API Key

После входа откройте личный кабинет, перейдите в **API Keys** и создайте новый ключ.

Перед API-вызовами создайте токен в AllRouter:

* (1) Откройте **Управление токенами**.
* (2) Нажмите **Добавить токен**.
* (3) Настройте имя, лимит, срок действия и ограничения.
* (4) Скопируйте сгенерированный ключ.

![](/docs_images/img_21.png)

Скопируйте данные API Keys.

![](/docs_images/img_22.png)

3. Шаг 3: настройте переменные окружения

Настройте переменные в macOS, Linux или Windows (ниже пример ручной настройки).

Пути файлов зависят от ОС. Убедитесь, что формат JSON корректен.

```bash
# Отредактируйте или создайте `settings.json`
# macOS & Linux: `~/.claude/settings.json`
# Windows: `<домашний-каталог>/.claude/settings.json`
# Добавьте или измените раздел `env`
# Замените `your_allrouter_api_key` на ваш API Key
```

```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "your_allrouter_api_key",
    "ANTHROPIC_BASE_URL": "https://allrouter.shengjian.net/v1",
    "API_TIMEOUT_MS": "3000000",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": 1
  }
}
```

```bash
# Затем отредактируйте или создайте `.claude.json`
# macOS & Linux: `~/.claude.json`
# Windows: `<домашний-каталог>/.claude.json`
# Добавьте `hasCompletedOnboarding`
```

```json
{
  "hasCompletedOnboarding": true
}
```

Результат конфигурации:

```env
ANTHROPIC_AUTH_TOKEN=API ключ токена
ANTHROPIC_BASE_URL=https://allrouter.shengjian.net/v1
ANTHROPIC_MODEL=Имя токена
```

Если появится сообщение **Do you want to use this API key**, выберите **Yes**.
